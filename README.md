# Sales Data Exploration

## Overview
This project explores and analyzes Superstore sales data to uncover key business insights such as top-selling products, regional performance, and sales trends over time.

## Objectives
- Identify the top 10 best-selling products
- Analyze sales and profit trends by month
- Compare performance across regions and product categories
- Derive key business insights from the data

## Skills & Tools Used
- **Python**: Data analysis using Pandas and visualization with Matplotlib & Seaborn
- **Google Colab**: For notebook execution
- **GitHub**: For version control and project sharing
